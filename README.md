# B-Store
Você é o tech lead de um squad e recebeu uma demanda prioritária do time de produto dizendo assim: "Squad! Precisamos criar um ecommerce para vender nossos produtos de crédito. Ele precisar ser escalável e de alta disponibilidade."

O time de arquitetura definiu alguns pontos que a solução precisa garantir para subir:

- Precisa ser documentada.
- Precisa rodar em container.
- Pode ser em qualquer linguagem ou framework.
- Precisa ter testes.
- Precisa ter um ambiente de desenvolvimento local de fácil inicialização.
- O provedor de pagamento ainda não está definido.
- A entrega deverá ser feita enviando um pull request para esse repositório.

Boa sorte.
